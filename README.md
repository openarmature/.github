# .github

This repository hosts organization-level GitHub configuration for
[OpenArmature](https://github.com/openarmature), including the org profile
README rendered at the org's landing page.

Looking for the framework itself? See
[openarmature-python](https://github.com/openarmature/openarmature-python)
for the Python reference implementation, or
[openarmature.org](https://openarmature.org) for the full specification
and documentation.

## Contents

- `profile/README.md` — rendered as the OpenArmature org's public landing
  page at github.com/openarmature.

## Contributing

Org profile changes should be proposed via pull request against
`profile/README.md`. Substantive changes to project positioning or
messaging should be discussed in an issue first.
